## Matplotlib ile Görselleþtirme 


Iris çiçeðine ait bilgilerin bulunduðu [Iris Dataseti](https://www.kaggle.com/kamrankausar/iris-data)  üzerinde Matplotlib kütüphenesi kullanýlarak görselleþtirmelerin yapýldýðý basit bir örnek.


###### Örneðin içeriði

* Datasetin okunmasý 
* Dataset içerisindeki bir sütunda kaç farklý deðer(Iris türünün) olduðunun öðrenilmesi
* Dataset içerisindeki sütun isimlerinin öðrenilmesi
* Dataset hakkýnda genel bilgilerin öðrenilmesi
* Her türün ayrý bir dataframe olarak ayrýþtýrýlmasý
* Türleri karþýlaþtýrmak için her tür için oluþturduðumuz dataframeler hakkýnda bilgi edinme.
* Matplotlib kütüphamesi kullanýlarak Line Plot ile  görselleþtirme 
* Matplotlib kütüphamesi kullanýlarak Scatter Plot ile  görselleþtirme
* Matplotlib kütüphamesi kullanýlarak Histogram ile  görselleþtirme
* Matplotlib kütüphamesi kullanýlarak Bar Plot ile  görselleþtirme
* Matplotlib kütüphamesi kullanýlarak Subplots ile  görselleþtirme
